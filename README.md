# shorin-dms-niri

开箱即用的Niri DMS预设


##  Usage 使用方法

- install安装

    ```
    yay -S shorin-dms-niri-git
    ```

    ```
    shorindms init 
    ```

    启动niri：
    
    ```
    niri-session
    ```

    如果你使用显示管理器的话在登录界面切换为niri

- update更新

    ```
    shorindms update
    ```
    以防万一，你的配置文件会被备份到`.cache`。

- uninstall卸载

    ```
    shorindms remove 
    ```

    ```
    yay -Rns shorin-dms-niri-git
    ```
